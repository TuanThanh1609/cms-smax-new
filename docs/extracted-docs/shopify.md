# 8.5.15. SHOPIFY

*Nguồn: [docs.smax.ai/vi/shopify](http://docs.smax.ai/vi/shopify)*

## 🛠 Quy trình tích hợp chi tiết

### 1. Tạo và lấy Access Token trên Shopify

* 
**Bước 1:** Tại trang quản trị Shopify nhấn **Settings**

<img src="./ui-tich-hop/shopify_img_f16712c1.png" alt="8.5.15. SHOPIFY" />

* 
**Bước 2:**   Đi đến mục **Apps and sales channels** và chọn **Develop apps**

<img src="./ui-tich-hop/shopify_img_534877f7.png" alt="8.5.15. SHOPIFY" />

* 
**Bước 3:** Click **Create an app**

<img src="./ui-tich-hop/shopify_img_942b6306.png" alt="8.5.15. SHOPIFY" />

* 
**Bước 4:** Nhập **App name** và nhấn **Create app**

<img src="./ui-tich-hop/shopify_img_cd9371c8.png" alt="8.5.15. SHOPIFY" />

* 
**Bước 5:** Chọn **Configure Admin API scopes**

<img src="./ui-tich-hop/shopify_img_76a412b0.png" alt="8.5.15. SHOPIFY" />

* 
**Bước 6:** Tích chọn các phạm vi truy cập (khuyến khích tích chọn tất cả) và nhấn **Save**

<img src="./ui-tich-hop/shopify_img_d2f5b1fb.png" alt="8.5.15. SHOPIFY" />

* 
**Bước 7:** Sau khi hiện thông báo **Configuration saved** nhấn **Install app**

<img src="./ui-tich-hop/shopify_img_e9ba1389.png" alt="8.5.15. SHOPIFY" />

* 
**Bước 8:** Một popup thông tin hiện lên nhấn **Install**

<img src="./ui-tich-hop/shopify_img_9fa1483b.png" alt="8.5.15. SHOPIFY" />

* 
**Bước 9:** Sao chép **Access token** tại đây

:::info
📌**Lưu ý:** Admin API access token **chỉ hiển thị 1 lần** vì lý do bảo mật trên Shopify, bạn cần sao chép và lưu lại để dùng sau này.

:::

<img src="./ui-tich-hop/shopify_img_1b98c1ab.png" alt="8.5.15. SHOPIFY" />

### 2. Kết nối Shopify vào SmaxAi

* **(1):** chọn nền tảng **Shopify**

* **(2)**: nhập **Tên**

* **(3)**: Click Active **Custom app**

* **(4)**: paste **Shop Name**

* **(5)**: **Access Token** và nhấn **Save**

<img src="./ui-tich-hop/shopify_img_e02f99c9.png" alt="8.5.15. SHOPIFY" />

**Lưu ý:** Tên cửa hàng được lấy tại URL [https://admin.shopify.com/store/**tencuahang**/](https://admin.shopify.com/store/**tencuahang**/)

<img src="./ui-tich-hop/shopify_img_1bff5873.png" alt="8.5.15. SHOPIFY" />
